# Real-Time-Chat-Application

use [React](https://react.dev/learn)

1. create project

```
npm / pnpm create vite@latest my-react-app -- --template reac
```

<u>my-react-app</u> is name of project

2. install tailwind css

```
pnpm add -D @tailwindcss/postcss autoprefixer
```
